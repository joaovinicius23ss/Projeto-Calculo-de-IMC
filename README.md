# Projeto-Calculo-de-IMC
Projeto feito em C# Windows Forms

Esse projeto é capaz de calcular o Indice de massa corporal de muitas pessoas utilizando o bloco de notas para armazenar os dados dos clientes, gerando um HTML geral de quantas pessoas sobre as pessoas e seus indices de massa corporal, tendo um CRUD como estrutura principal.

Esse projeto foi pensado para Clínicas que nessecitam de um gerenciamento dos dados dos clientes melhor e mais pratico.

Quando você for baixar esse zip, lembre-se de trocar o local onde o bloco de notas e o html serão criados
